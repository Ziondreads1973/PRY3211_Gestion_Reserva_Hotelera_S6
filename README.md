# Hotel Pacific Reef - MVP Semana 7

Proyecto desarrollado para la asignatura PRY3211 - Ingeniería de Software.

## Descripción

Este repositorio contiene el avance funcional de Semana 7 del Sistema de Gestión de Reserva Hotelera para Hotel Pacific Reef.  
El objetivo del MVP es demostrar la integración entre front-end, back-end y base de datos en un ambiente de prueba.

## Tecnologías utilizadas

- Python
- Flask
- SQLite
- HTML
- CSS

## Funcionalidades implementadas

- Página principal del MVP.
- Consulta de disponibilidad de habitaciones.
- Listado de habitaciones disponibles.
- Registro básico de reserva.
- Cálculo automático de abono del 30%.
- Confirmación de reserva.
- Panel administrador para visualizar reservas registradas.
- Verificación de conexión con la base de datos.
