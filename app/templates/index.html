Release Semana 8 · Prototipo funcional integrado

Este prototipo corresponde al release funcional de Semana 8 del sistema Hotel Pacific Reef. El proyecto se basa en el MVP construido en Semana 7 y actualmente permite demostrar la integración entre front-end, back-end y base de datos SQLite en un ambiente local de pruebas.

El flujo principal permite consultar disponibilidad de habitaciones, registrar una reserva, calcular automáticamente los montos asociados y visualizar las reservas registradas desde un panel administrador.

Consultar disponibilidad

Funcionalidades disponibles

  • Consulta de disponibilidad de habitaciones.
  • Validación de fechas de entrada y salida.
  • Listado de habitaciones disponibles según capacidad.
  • Registro básico de reserva desde interfaz web.
  • Cálculo automático del total de reserva.
  • Cálculo automático de abono requerido del 30%.
  • Cálculo automático del saldo pendiente.
  • Confirmación de reserva con código generado.
  • Panel administrador para visualizar reservas registradas.
  • Verificación de conexión con base de datos SQLite.

Estado del release

El sistema se encuentra en ambiente de prueba local. Durante Semana 8 se continuará con la incorporación de mejoras funcionales, pruebas documentadas, actualización del DOD, Manual de Usuario Testing y evidencias del sprint.

Las próximas mejoras planificadas consideran la integración con un servicio web externo y la disponibilidad de recursos mediante endpoints JSON para consumo por otras aplicaciones.
